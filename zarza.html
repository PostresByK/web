Cheescake de zarzamora

Ingredientes

  • 150 Gramos Mantequilla a temperatura ambiente, para la base
  • 1 Lata Leche Condensada LA LECHERA® para la base
  • 2 Piezas Yema para la base
  • 3 1/2 Tazas Harina de trigo para la base
  • 4 Paquetes Queso crema en trozos
  • 1 Cucharadita Polvo para hornear para la base
  • 4 Piezas Huevo
  • 1/2 Taza Mermelada zarzamora



Cómo hacer Cheescake de zarzamora:

  1. Horno precalentado a 180° C
  2. Para la base bate la mantequilla hasta acremar, agrega la Leche Condensada LA LECHERA® y las yemas, mezcla hasta integrar completamente. Añade la harina preveiamente cernida con el polvo para hornaear hasta obtener una masa tersa. Vuelca a una mesa previamente enharinada y amasa ligeramente. Extiende con la ayuda de un rodillo, cubre un molde para pay previamente engrasado, recorta las orillas y reserva.
  3. Bate los paquetes de queso rema con la leche Condensada LA LECHERA® y los huevos. Vierte sobre la base para pay. Hornea 55 minutos, retira del horno, deja enfriar, decora con la mermelada.
